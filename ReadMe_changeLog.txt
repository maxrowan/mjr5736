A. To run:
	1. Ensure Node is installed.
	
	2. Open a command prompt and navigate to the directory the files are in.
	
	3. Run the commands:
		> npm install twitter
		> npm install express
		> npm install socket.io
		
	4. To launch, run the command:
		> node server.js

B. Changes:
	1. Map is now a heatmap. Tweet locations are no longer represented by pins.
	
	2. Latitude and Longitude are fixed. They were reversed in the last iteration.
